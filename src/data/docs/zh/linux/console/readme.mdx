### Terminal: Paste

在 Bash 中，Ctrl + V 用于插入下一个按键的字面值。这在需要输入特殊字符或控制字符时非常有用。例如，如果你想输入一个实际的 Ctrl + C 字符而不是执行中断命令，你可以按 Ctrl + V 然后按 Ctrl + C。

1. 打开 `@/.inputrc` 文件（如果文件不存在，可以创建一个）：

```sh
code @/.inputrc
```

2. 在文件中添加以下内容：

```sh
"\C-v": paste-from-clipboard
```

3. 保存并关闭文件。

4. 重新启动 Visual Studio Code 或在终端中运行以下命令以使更改生效：

```sh
bind -f @/.inputrc
```

请注意，这个方法依赖于你的终端模拟器支持 `paste-from-clipboard` 功能。如果你的终端模拟器不支持这个功能，你可能需要使用 Visual Studio Code 本身的设置来更改粘贴快捷键。

在 Visual Studio Code 中，你可以通过以下步骤更改终端的粘贴快捷键：

1. 打开 Visual Studio Code。
2. 进入设置：文件 > 首选项 > 键盘快捷方式。
3. 搜索 terminal paste。
4. 找到 Terminal: Paste 命令，点击右侧的铅笔图标，然后按 Ctrl + V 进行绑定。

这样，你就可以在 Visual Studio Code 的终端中使用 Ctrl + V 进行粘贴了。
