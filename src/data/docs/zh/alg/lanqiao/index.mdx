## 竞赛规则

### 编程环境

#### python
- python 3.8.6
- IDLE (python 自带编辑器)

```sh
# 启动 IDLE
idle

python -m idlelib
```

### OI赛制
1. 无现场提交反馈. 也就是说, 你提交了代码, 你不知道对不对, 只有等到比赛结束, 才能知道对不对.
2. 按点给分. 根据测试点通过的数量给分. 
  e.g:
    - $对于 30\% 的评测用例, 1\leq n\leq 10^3, 1\leq a_i\leq 10^3$
    - $对于 100\% 的评测用例, 1\leq n\leq 2\cdot10^5, 1\leq a_i\leq 10^{16}$

### 结果填空题

可以用别的语言, 工具, 或手算

### 编程大题

通过测试用例估算正解的时间复杂度, 一般要在 $n^7,n^8$ 以内, 例如用例是 
- $n\leq 10^3$, 那么时间复杂度 可以选 $O(n^2)$,
- $n\leq 10^5$, 那么时间复杂度 可以选 $O(n\log n)$

### input

```py
import sys
input = lambda: sys.stdin.readline().strip()
s = input()
# 读取一行  -----------------------------------------
# 1 2 3
a, b, c = map(int, input().split())
# 读取多行 -----------------------------------------
# 2
# 1
# 3
# lst = [1, 3]
n = int(input())
lst = [int(input()) for _ in range(n)]

# 矩阵读取 -----------------------------------------
# 2 3
# 1 2 3
# 4 5 6
# matrix = [[1, 2, 3], [4, 5, 6]]
n, m = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(n)]
print(matrix)

# 读取多行
lines = sys.stdin.readlines()
```
